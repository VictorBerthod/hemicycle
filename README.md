# hemicycle

Site web educatif rendant lisibles les structures et mouvements de pouvoir a l'Assemblee nationale francaise.

## Stack

- **Backend**: Python + FastAPI + SQLite
- **Frontend**: SvelteKit + D3.js
- **Data**: API Assemblee nationale, HATVP, NosDéputés.fr
- **Infra**: Docker Compose

## Development

```bash
docker compose up -d
```
