# Fiche mémo — Les institutions françaises

*Aide-mémoire synthétique de la Ve République*

---

## La République en un coup d'œil

**Régime :** République semi-présidentielle de la Ve République, Constitution du 4 octobre 1958.

**Devise :** Liberté, Égalité, Fraternité — République indivisible, laïque, démocratique et sociale, organisée de manière décentralisée.

**Trois pouvoirs :** Exécutif (Président + Gouvernement) — Législatif (Parlement bicaméral) — Juridictionnel (juge judiciaire, juge administratif, Conseil constitutionnel).

## Les institutions en fiches

### PRÉSIDENT DE LA RÉPUBLIQUE

| | |
|---|---|
| **Élection** | Suffrage universel direct à 2 tours |
| **Mandat** | 5 ans, renouvelable une fois consécutivement |
| **Siège** | Palais de l'Élysée (Paris) |
| **Rôle** | Chef de l'État, garant des institutions, chef des armées, conduit la diplomatie |
| **Pouvoirs propres** | Nomme le Premier ministre, dissout l'Assemblée nationale, déclenche le référendum (art. 11), exerce les pouvoirs exceptionnels (art. 16) |
| **Pouvoirs partagés** | Promulgue les lois, signe les ordonnances et décrets, nomme aux hauts emplois, ratifie les traités |
| **Responsabilité** | Irresponsable politiquement ; destitution possible par la Haute Cour pour manquement grave |

### PREMIER MINISTRE

| | |
|---|---|
| **Nomination** | Par le Président de la République |
| **Siège** | Hôtel de Matignon (Paris) |
| **Rôle** | Dirige l'action du Gouvernement, exécute les lois, exerce le pouvoir réglementaire |
| **Pouvoirs clés** | Engage la responsabilité du Gouvernement (art. 49), peut recourir à l'article 49.3, dispose de l'administration |
| **Responsabilité** | Devant l'Assemblée nationale uniquement |
| **Titulaire actuel** | Sébastien Lecornu (depuis le 9 septembre 2025) |

### ASSEMBLÉE NATIONALE

| | |
|---|---|
| **Composition** | 577 députés |
| **Élection** | Suffrage universel direct, scrutin majoritaire à 2 tours |
| **Mandat** | 5 ans (sauf dissolution) |
| **Siège** | Palais Bourbon (Paris) |
| **Pouvoirs** | Vote la loi et le budget, contrôle le Gouvernement, peut le renverser (motion de censure) |
| **Particularité** | A le dernier mot en cas de désaccord avec le Sénat ; saisie en premier des projets de loi de finances |

### SÉNAT

| | |
|---|---|
| **Composition** | 348 sénateurs |
| **Élection** | Suffrage universel indirect (grands électeurs) |
| **Mandat** | 6 ans, renouvelable par moitié tous les 3 ans |
| **Siège** | Palais du Luxembourg (Paris) |
| **Pouvoirs** | Vote la loi, contrôle le Gouvernement (mais ne peut pas le renverser), évalue les politiques publiques |
| **Particularités** | Ne peut être dissous ; représente les collectivités territoriales ; son président assure l'intérim présidentiel en cas de vacance |

### CONSEIL CONSTITUTIONNEL

| | |
|---|---|
| **Composition** | 9 membres + anciens présidents de la République (membres de droit) |
| **Désignation** | 3 par le Président, 3 par le président de l'Assemblée, 3 par le président du Sénat |
| **Mandat** | 9 ans, non renouvelable, renouvellement par tiers |
| **Siège** | Palais-Royal (Paris) |
| **Rôle** | Contrôle la conformité des lois à la Constitution (a priori et a posteriori via QPC) ; juge des élections nationales |
| **Saisine** | Président, Premier ministre, présidents des chambres, 60 députés ou 60 sénateurs ; QPC via tout justiciable |

### CONSEIL D'ÉTAT

| | |
|---|---|
| **Création** | 1799 (Napoléon Bonaparte) |
| **Siège** | Palais-Royal (Paris) |
| **Double rôle** | Conseiller du Gouvernement (avis sur les projets de loi et décrets) et juge administratif suprême |
| **Compétences juridictionnelles** | Recours pour excès de pouvoir contre les décrets, cassation des arrêts des cours administratives d'appel |

### COUR DE CASSATION

| | |
|---|---|
| **Rôle** | Juridiction suprême de l'ordre judiciaire |
| **Mission** | Ne rejuge pas les faits ; vérifie la bonne application du droit par les juridictions du fond |
| **Siège** | Palais de Justice (Paris) |

### CONSEIL ÉCONOMIQUE, SOCIAL ET ENVIRONNEMENTAL (CESE)

| | |
|---|---|
| **Composition** | 175 membres représentant le monde économique, social et environnemental |
| **Mandat** | 5 ans |
| **Siège** | Palais d'Iéna (Paris) |
| **Rôle** | Troisième assemblée constitutionnelle, consultative ; donne des avis sur les sujets économiques, sociaux et environnementaux |
| **Saisine citoyenne** | Pétition de 150 000 signatures minimum |

### COUR DES COMPTES

| | |
|---|---|
| **Rôle** | Contrôle des finances publiques (État, opérateurs, Sécurité sociale) |
| **Mission** | Assiste le Parlement et le Gouvernement, évalue les politiques publiques, certifie les comptes de l'État |
| **Siège** | Rue Cambon (Paris) |
| **Démembrement** | Chambres régionales des comptes (CRC) pour les collectivités territoriales |

### DÉFENSEUR DES DROITS

| | |
|---|---|
| **Statut** | Autorité constitutionnelle indépendante (art. 71-1), créée en 2008 |
| **Nomination** | Par le Président de la République, pour 6 ans non renouvelables |
| **Domaines** | Relations avec les services publics, droits de l'enfant, lutte contre les discriminations, déontologie de la sécurité, lanceurs d'alerte |
| **Saisine** | Directe et gratuite par tout citoyen |

## Procédures essentielles

### Vote d'une loi (procédure ordinaire)

1. **Initiative** — projet de loi (Gouvernement) ou proposition de loi (parlementaires).
2. **Dépôt et examen en commission** dans la première chambre saisie.
3. **Vote en séance plénière** (article par article, puis sur l'ensemble).
4. **Navette** vers l'autre chambre, qui peut amender.
5. **Commission mixte paritaire (CMP)** en cas de désaccord persistant.
6. **Dernier mot** à l'Assemblée nationale si la CMP échoue (sauf lois constitutionnelles).
7. **Conseil constitutionnel** (saisine facultative pour les lois ordinaires).
8. **Promulgation** par le Président dans les 15 jours.
9. **Publication** au Journal officiel.

### Motion de censure (art. 49 alinéa 2)

- Déposée par au moins 1/10e des députés (58).
- Vote 48 heures après le dépôt.
- Adoptée si majorité absolue des députés (289).
- Conséquence : démission du Gouvernement.

### Article 49 alinéa 3

- Le Premier ministre engage la responsabilité du Gouvernement sur un texte.
- Le texte est réputé adopté sans vote, sauf adoption d'une motion de censure dans les 24 h.
- Limité à un texte par session ordinaire (sauf lois de finances et de financement).

### Question prioritaire de constitutionnalité (QPC)

- Soulevée par un justiciable à l'occasion d'un procès.
- Filtrage par la Cour de cassation ou le Conseil d'État.
- Examen par le Conseil constitutionnel dans un délai de 3 mois.
- Si la loi est jugée inconstitutionnelle, elle est abrogée.

## Qui fait quoi ? — Synthèse

| Action | Qui est compétent |
|---|---|
| Voter les lois | Parlement (Assemblée nationale + Sénat) |
| Appliquer les lois | Gouvernement, via décrets et arrêtés |
| Veiller à la conformité des lois à la Constitution | Conseil constitutionnel |
| Juger les administrations | Juridictions administratives (jusqu'au Conseil d'État) |
| Juger les particuliers (civil et pénal) | Juridictions judiciaires (jusqu'à la Cour de cassation) |
| Élire le Président de la République | Tous les citoyens (suffrage universel direct) |
| Élire les députés | Tous les citoyens (suffrage universel direct) |
| Élire les sénateurs | Les grands électeurs (suffrage universel indirect) |
| Gérer les écoles primaires | La commune |
| Gérer les collèges | Le département |
| Gérer les lycées | La région |
| Gérer les universités | L'État |
| Délivrer un permis de construire | Le maire (commune) |
| Payer le RSA | Le département |
| Représenter l'État localement | Le préfet |
| Contrôler les finances publiques | Cour des comptes (et chambres régionales) |
| Protéger les données personnelles | CNIL |
| Réguler l'audiovisuel et le numérique | Arcom |
| Aider en cas de difficulté avec un service public | Défenseur des droits |

- **Monnaie :** l'euro (€), depuis 2002.
- **Fête nationale :** le 14 juillet.
