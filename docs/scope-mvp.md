# Perimetre MVP vs etendu

## MVP — "L'Assemblee mise a nu"

| Fonctionnalite | Description |
|----------------|-------------|
| Fiches deputes | Profil, groupe, votes cles, declarations HATVP |
| Scrutins publics | Tous les votes avec resultat et position par depute |
| Visualisation votes | Graphe simple : qui vote avec qui, dissidences |
| Angles editoriaux | 10 analyses generees par IA, validees humainement |
| Sourcing transparent | Chaque donnee cliquable vers la source officielle |
| Recherche | Par depute, par theme, par scrutin |

## Perimetre etendu (post-MVP)

- Senat + Parlement europeen
- Graphes d'influence (lobbys via HATVP)
- Suivi des promesses vs votes
- Comparateur de deputes
- Alertes personnalisees ("ton depute a vote X")
- Niveau 3 d'analyse (patterns, incoherences automatiques)
- API publique pour reutilisation
- Partage social optimise (OG images generees)

## Publics cibles

| Priorite | Public | Besoin |
|----------|--------|--------|
| Primaire | Citoyens 18-35 ans, politises mais pas experts | Comprendre vite, partager |
| Secondaire | Journalistes, militants associatifs, enseignants | Donnees fiables, reutilisables |
| Tertiaire | Chercheurs en science politique | Credibilite methodologique |
